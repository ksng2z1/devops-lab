# DevOps Lab

Мой учебный репозиторий по пути в Junior DevOps.

## Сервер

- **Hostname:** labvm
- **OS:** Ubuntu 24.04 LTS
- **Роль:** учебная VM в Hyper-V
- **IP:** 192.168.1.60 (статика через netplan + networkd)

## Структура

- `configs/` — копии серверных конфигов (netplan, ufw, motd)
- `scripts/` — bash-скрипты мониторинга и автоматизации
- `dayNN/` — материалы дней обучения
